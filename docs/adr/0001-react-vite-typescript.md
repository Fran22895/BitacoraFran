# ADR 0001: React, Vite Y TypeScript

## Estado

Aceptada.

## Contexto

La app necesita una base moderna, rapida en desarrollo y mantenible para formularios, mapas, calendario y PDF.

## Decision

Usar React + Vite + TypeScript.

## Consecuencias

- Desarrollo rapido con HMR.
- Tipado compartido para dominio, formularios y tests.
- Buen encaje con Supabase y librerias del ecosistema React.
