# Roadmap

## V1 actual

- Aplicacion funcional en modo demo local.
- Auth Google preparada con Supabase.
- CRUD de viajes y modulos principales.
- Dashboard, filtros, detalle, calendario, mapa, PDF y modo oscuro.
- Documentacion y SQL inicial.

## V1.1

- Sustituir store local por TanStack Query + Supabase real.
- Subida real de documentos e imagenes a Supabase Storage.
- Invitaciones por email para miembros.
- Confirmaciones antes de eliminar.
- Mejoras de accesibilidad en formularios largos.

## V1.2

- Exportacion PDF mas visual.
- Vista de presupuesto avanzada por categoria.
- Historial de cambios por viaje.
- Busqueda global por documento, contacto y reserva.

## Futuro

- Cambio de divisas opcional mediante API externa.
- Modo offline con sincronizacion posterior.
- Plantillas de viaje.
- Importar reservas desde archivos.
